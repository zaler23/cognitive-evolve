# Support

## Project status

CognitiveEvolve is a pre-release source project and protocol workbench. It is a
source-installable beta for local CLI/API use, not a packaged public installer
or hosted service.

## Where to get help

- Use GitHub issues for reproducible bugs, documentation problems, and focused feature requests.
- Use the design review issue template for architecture, adapter, eval, memory, or workflow proposals.
- Do not use public issues for vulnerabilities; follow `SECURITY.md`.

## What to include

For bugs, include:

- Command run.
- Expected behavior.
- Actual behavior.
- Relevant sanitized logs.
- Python version and operating system.
- Whether a standalone runtime was involved.

For feature requests, include:

- Problem statement.
- Why existing native capabilities are insufficient.
- Complexity budget and validation signal.

## Out of scope

- Packaged installer, PyPI release, or hosted-service support before the release boundary is accepted.
- Help recovering leaked secrets in public issues.
- Hosted-service or credential setup unless an accepted adapter decision exists.
